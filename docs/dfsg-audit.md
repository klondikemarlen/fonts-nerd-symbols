# DFSG glyph provenance audit

Track this before Debian/Ubuntu archive sponsorship. Symbols Only is a combined Nerd Fonts font; do not claim DFSG compliance until every bundled glyph source is audited or excluded.

Issue: https://github.com/klondikemarlen/fonts-nerd-symbols/issues/1

## Required outcome

For each included glyph set, record:

- upstream source URL;
- preferred form for modification;
- license text and copyright holders;
- whether the license is DFSG-free for Debian `main`;
- whether any logo/trademark terms affect redistribution or modification;
- package decision: keep, exclude, or replace with a Debian-packaged font dependency.

If any glyph set is doubtful, rebuild or repack Symbols Only without it before sponsorship.
